## 二分探索による解法

### 1回目 (8m52s)
* 時間計算量: O(N * log(high - low))
* 空間計算量: O(1)
N: len(weights)

* 最近(1ヶ月前)に解いていたため、ロジック自体は頭に入っていた。

```python
class Solution:
    def shipWithinDays(self, weights: List[int], days: int) -> int:
        def is_shipped_with_capacity(capacity: int) -> bool:
            total_weight = 0
            days_so_far = 0
            for weight in weights:
                if total_weight + weight > capacity:
                    total_weight = 0
                    days_so_far += 1

                total_weight += weight
                
                if days_so_far > days:
                    return False
                
            return True if days_so_far < days else False

        low = max(weights)
        high = sum(weights)

        while low < high:
            mid = (low + high) // 2
            if is_shipped_with_capacity(mid) == True:
                high = mid
            else:
                low = mid + 1
        return low
```

## 全探索による解法 (TLE)
* 時間計算量: O(N * (high - low))
* 空間計算量: O(1)
```python
class Solution:
    def shipWithinDays(self, weights: List[int], days: int) -> int:
        def is_shipped_in_capacity(capacity: int) -> bool:
            total_weight = 0
            days_so_far = 1
            for weight in weights:
                if total_weight + weight > capacity:
                    days_so_far += 1
                    total_weight = 0
                total_weight += weight

                if days_so_far > days:
                    return False

            return True

        low = max(weights)
        high = sum(weights)

        for cap in range(low, high + 1):
            if is_shipped_in_capacity(cap):
                return cap
```
