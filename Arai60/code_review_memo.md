- スネーク: 単語をアンダースコア（_）で区切り、全ての文字を小文字で書く方式
- チョムスキー階層: 
- フロイドのうさぎとかめ: あるリスト内の2つのポインタ（ひとつは遅いポインタ、もうひとつは速いポインタ）を使用して、リスト内のループやサイクルを検出する方法
