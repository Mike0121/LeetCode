## two-pointerによる解法
---
時間計算量: O(len(t))<br>
空間計算量: O(1)

### 1回目 (3m20s)
```python
class Solution:
    def isSubsequence(self, s: str, t: str) -> bool:
        if s == "":
            return True
        s_index = 0

        for t_index in range(len(t)):
            if s[s_index] == t[t_index]:
                s_index += 1

            if s_index == len(s):
                return True
        return False
```


### 2回目
continueによる書き換え。
t_indexは利用せず、```python for t_letter in t```として直接文字と比較しても良いかと思った。

```python
class Solution:
    def isSubsequence(self, s: str, t: str) -> bool:
        if s == "":
            return True
        s_index = 0
        for t_index in range(len(t)):
            if s[s_index] != t[t_index]:
                continue
            
            s_index += 1
            if s_index == len(s):
                return True
        return False
```

## stackによる解法 (2m20s)
---
時間計算量: O(len(s) + len(t))<br>
空間計算量: O(len(s))<br>

```python
class Solution:
    def isSubsequence(self, s: str, t: str) -> bool:
        if s == "":
            return True
        s_letters = deque(s)

        for t_letter in t:
            if s_letters[0] == t_letter:
                s_letters.popleft()

            if not s_letters:
                return True
        return False
```

## 正規表現による解法
---
https://docs.python.org/ja/3/library/re.html#re.match

```python
class Solution:
    def isSubsequence(self, s: str, t: str) -> bool:
        pattern = ""
        for letter in s:
            pattern += ".*" + letter
        return re.match(pattern, t) != None
```
