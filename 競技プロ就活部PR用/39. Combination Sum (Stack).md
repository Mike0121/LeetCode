### 39. Combination Sum Stackを利用した解法
```
再帰が深いので、半分はループで書く、または全部ループで書けますか。
全部ループというのは、スタックを使います。スタックに、全部の引数のペアを積んで、これを処理すること、とします。
```

```python
# 最悪時間計算量: O(len(candidates)^(target//min(candidates)+1))
# 最悪空間計算量: O(target / min(candidtes) + all_valid_combinationsのストック分)
class Solution:
    def combinationSum(self, candidates: List[int], target: int) -> List[List[int]]:
        all_valid_combinations = []
        stack = [(0, target, [])]
        while stack:
            index, current_target, current_combination = stack.pop()
            
            if index >= len(candidates):
                if current_target == 0:
                    all_valid_combinations.append(current_combination)
                continue
            if candidates[index] <= current_target:
                stack.append((index, current_target - candidates[index], current_combination + [candidates[index]]))
                
            stack.append((index + 1, current_target, current_combination))

        return all_valid_combinations
```
