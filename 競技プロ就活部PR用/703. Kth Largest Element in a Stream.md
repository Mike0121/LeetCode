### 1回目 (12m25s)
---
時間計算量: O(k + (N - k) \* logk) (init), O(logk) (add)<br>
空間計算量: O(k)

```python
class KthLargest:
    def __init__(self, k: int, nums: List[int]):
        self.k = k
        self.nums_kth = nums[:k]
        heapq.heapify(self.nums_kth)

        for i in range(k, len(nums)):
            if nums[i] > self.nums_kth[0]:
                heapq.heappushpop(self.nums_kth, nums[i])
        
    def add(self, val: int) -> int:
        if len(self.nums_kth) < self.k:
            heapq.heappush(self.nums_kth, val)
        elif self.nums_kth[0] < val:
            heapq.heappushpop(self.nums_kth, val)

        return self.nums_kth[0]
```
